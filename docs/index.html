Reflexiones

Semana 1 - Introduccion y Uso de Visual Studio Code

Aprendi a manejar Visual Studio Code como entorno de desarrollo principal, aprovechando el plugin Emmet para escribir codigo HTML y CSS mas rapido. El uso de atajos de teclado mejoro significativamente mi productividad, permitiendome trabajar de forma mas fluida y profesional.

Semana 2 - HTML y CSS: Estructuras y Estilos

Durante esta semana consolide mis conocimientos de HTML5 y CSS3, creando paginas web estructuradas, limpias y visualmente agradables. Aprendi a implementar disenos responsive, aplicar animaciones CSS y mejorar la experiencia visual de las interfaces, base esencial para integrar frameworks mas avanzados.

Semana 3 - Frameworks CSS: Bootstrap y Tailwind

Trabaje con Bootstrap y Tailwind CSS, dos frameworks que facilitan el diseno web profesional. Con Bootstrap entendi el sistema de grillas y componentes predefinidos, mientras que con Tailwind domine las clases utilitarias para personalizar cada elemento sin escribir CSS adicional. La practica incluyo cards animadas, formularios, sliders RGB y galeria responsive.

Semana 4 - JavaScript: Ruleta y Sorteo de Equipos

Desarrolle una ruleta interactiva controlada por eventos de teclado, con animaciones dinamicas, colores aleatorios y sorteo automatico de equipos. Aprendi a manipular el DOM, crear y modificar elementos en tiempo real y organizar funciones de JavaScript de forma modular.

Semana 5 - Instalacion de Vite, React y Next.js

Explore el entorno de Vite para crear proyectos React y Next.js con alto rendimiento. Comprendi la estructura base de un proyecto React y las diferencias entre React tradicional y Next.js, especialmente en la generacion estatica y el renderizado del lado del servidor.

Semana 6 - Fundamentos y Componentes de React

Profundice en la creacion de componentes funcionales, el uso de props, children y la comunicacion entre componentes.

  • Aplicacion con siete componentes responsivos.
  • Componentes anidados que muestran objetos de datos.
  • Comunicacion padre e hijo gestionada con useState.
  • Renderizado de listas utilizando el metodo map.
